# 安装方法 Windows

```
$ git clone https://github.com/PythonerKK/eleme-api-by-django-restful-framework.git
$ cd eleme-api-by-django-restful-framework
$ virtualenv Venv
$ cd Venv/Scripts
$ activate
$ pip install -r requirements.txt
```

# 安装方法 Mac

```
$ git clone https://github.com/PythonerKK/eleme-api-by-django-restful-framework.git
$ cd eleme-api-by-django-restful-framework
$ virtualenv Venv
$ source Venv/bin/activate
$ pip install -r requirements.txt
```
